Photo 제작순서
- 요구사항
-목록조회 : 4열 그리드 화면으로 목록조회
-사진추가 : 추가 버튼으로 제목, 설명, 이미지 파일 선택 후 추가
- 사진 상세조회 : 목록에서 사진 클릭하면 나오게
		필요한 데이터 : 타이틀 , 설명, 이미지
		기능 : 삭제 , 수정, 목록버튼
- 사진삭제 : 상세조회화면에서 삭제버튼을 클릭하면 삭제
- 사진수정 : 상세조회화면에서 수정버튼을 누르면 수정화면으로 이동
	타이틀, 설명 ,이미지 수정, 이미지 프리뷰

- 구현
    -  메뉴목록에 photos 추가, photo 목록 컴포넌트를 출력
        - 컴포넌트 생성, 라우터 연결

    - 리덕스 스토어에 넣을 스테이트를 만들기 
        - 포토 슬라이스 생성, 데이터타입 설계, 기본값, reducer export
        - 스토어에서 임포트
        - 초기값 (목록 데이터) 넣기, photo목록 컴포넌트에서 출력

    - photo 목록 컴포넌트 JSX 구현
        - 글로벌 스테이트에서 데이터 받아서 화면에 구현

    - photo Create 컴포넌트 구현
        - App.tsx에서 path 연결
        - 추가버튼 클릭시 생성되는 data를 dispatch

    - 추가기능 만들기 1 
        - 추가버튼 만들고 추가화면 이동 추가화면 만들기

    - 추가기능 만들기 2
        - 입력폼 및 버튼 만들기

    - 추가기능 만들기 3
        - 입력폼에 ref 객체 연결 및 ref 값 받아오기

    - 추가기능 만들기 4
        - ref.current 데이터를 기준으로 데이터 객체(redux state구조)를 생성

    - 추가기능 만들기 5
        - 데이터 객체를 redux state에 추가(dispatch(action함수))
           -> reducer  함수를 생성하고, action 을 exprot 해야함
        - 목록화면으로 이동

    - 추가기능 만들기 6
        - 목록 버튼을 클릭하면 목록화면으로 넘어가기
