Flux는 Facebook에서 클라이언트-사이드 웹 어플리케이션을 만들기 위해 
사용하는 어플리케이션 아키텍쳐다. 단방향 데이터 흐름을 활용해 뷰 
컴포넌트를 구성하는 React를 보완하는 역할을 한다.
-> Flux 클라이언트 데이터 관리 아키텍처
-> 단방향 데이터 흐름
-----------------------------------------------------------------------------------
flux: 클라이언트 사이트 데이터 관리 아키텍쳐(큰규모 구조) 
                                               디자인 (작은규모 구조)
    -> 추상적인, 이론적인, 원리적인
    -> Veiw는 Store를 조회
    -> View에서 Action객체 Dispatcher 넘기면 
               Dispatcher가 Store를 변경

Redux: flux 패턴을 라이브러리 (js/ts) 만들어 놓은것
    -> ActionCreator: 액션객체를 생성하는 함수
    -> Action 액션타입 (명령) , 페이로드(매개변수)
    -> Dispatcher 액션타입과 페이로드를 받아서 리듀서를 실행
    -> Reducer 기존 state아 액션의 페이로드 state를 변경

React-Redux: React 에서 Redux를 쓸수 있게 만들어 놓은것
